## App Store Optimization (ASO) Tips

### Title Optimization

- Keep app name short and memorable: "Plena"
- Subtitle should include key search terms: "meditation" and "biometrics"

### Description Optimization

- First 3 lines are most important (visible in preview)
- Include keywords naturally
- Use emojis sparingly (they don't help search)
- Highlight unique features (stress zones, HRV tracking)

### Screenshot Strategy

4. **Screenshot 4:** Data visualization charts
